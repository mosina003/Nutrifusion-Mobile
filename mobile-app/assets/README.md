# Assets Placeholder

This directory will contain app assets such as:

- `icon.png` - App icon (1024x1024px)
- `splash.png` - Splash screen image (1242x2436px)
- `adaptive-icon.png` - Android adaptive icon (1024x1024px)
- `favicon.png` - Web favicon (48x48px)

## Temporary Setup

For now, the app will use default Expo assets. To customize:

1. Create your icon and splash images
2. Place them in this directory
3. Update `app.json` with the correct paths

## Recommended Sizes

- **icon.png**: 1024x1024px (PNG with transparency)
- **splash.png**: 1242x2436px (PNG)
- **adaptive-icon.png**: 1024x1024px (PNG with transparency)
- **favicon.png**: 48x48px (PNG)

You can generate these using tools like:
- [Figma](https://www.figma.com/)
- [Canva](https://www.canva.com/)
- [MakeAppIcon](https://makeappicon.com/)
